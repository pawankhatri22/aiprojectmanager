# Logical Architecture

Angular Frontend -> REST API -> Spring Boot Backend -> Service Layer -> Repository Layer (JPA) -> H2 DB

Backend modules:
- Auth Module
- User Module
- Profile Module
- Session Module
- Messaging Module
- Payment Module
- Admin Module

# Recommended Build Order
1. Project setup
2. Auth + JWT + User
3. Profiles
4. Mentor listing/search
5. Session booking
6. Messaging
7. Payment simulation
8. Admin dashboards
9. UI polishing
